### 3. **Dynamic Zero Documentation**
**URL**: https://autoppia.substack.com/p/dynamic-zero-the-overfitting-punisher

**Key Information**:
- **Dynamic IWA (D1-D4)**: Ever-changing web environments
- **Winner Takes All (WTA)**: Only top miner gets full reward
- **Time Factor Removed**: No longer part of scoring
- **Efficiency Rewards Removed**: No longer reward fewer actions
- **Focus**: Task completion + precision only

**Use For**:
- Understanding scoring system
- Building agents that adapt (not memorize)
- Understanding competition structure
